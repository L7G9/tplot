    Introduction

    Each event in a timeline occurs at a given time set by the user, the app uses this time and the scale settings to position the event accurately on the scale of a PDF timeline.

    Each event has the following in common...

    • Start Time
    • Has End
    • End Time
    • Title
    • Description
    • Event Area
    • Tags

    Start Time

    The time at which the event occurred or started. Because each type of timeline uses different time units, this setting will vary between the different timeline types. On a PDF timeline, an event with a start and no end time will start at the appropriate position on the scale and take up enough room to hold the description of the event only. The start time is the first line describing the event.

    Has End

    Events can have an end time which is enabled by this option.

    End Time

    The time at which the event ended. Because each type of timeline uses different time units, this setting will vary between the different timeline types. On a PDF timeline, an event with a start and end time will start and finish at the appropriate position on the scale.

    Title

    A title which is used to identify the event in TPlot and displayed as a heading of the event when it is displayed on a PDF timeline.

    Description

    An optional description which can be used to add additional details about the event.

    Event Area

    Each event can have a single event area, which is used to position groups of events on a PDF timeline. An event must have an event area otherwise it won't be displayed on a PDF timeline.

    See the reference guide for event areas for more details.

    Tags

    Each event can have multiple tags.

    See the reference guide for tags for more details.
