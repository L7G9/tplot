Introduction

A timeline's scale and each event area has a page position setting, these are used to control where on the page they are in relation to each other. The scale page position can be set when creating a new timeline or from timeline settings.

An event area page position can be set when creating a new event area or by editing the event area.

Example 1

The scale’s page position is 0, event area A’s position is 1 and event area B’s position 2.

When landscape, the scale will be at the bottom of the page, event area A will be above the scale and event area B will be above A.

When portrait, the scale will be on the left hand side of the page, event area A will be to the right of the scale and event area B will be to the right of A.

Example 2

The scale’s page position is 1, event area A’s position is 0 and event area B’s position 2.

When landscape, event area A will be at the bottom of the page, the scale will be above event area A and event area B will be above the scale.

When portrait, event area A will be on the left hand side of the page, the scale will be to the right of event area A and event area B will be to the right of the scale.
