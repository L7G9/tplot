Introduction

This guide shows what effect the orientation & page size settings have on the PDF timelines timelines created by TPlot.

Orientation

A timeline can have a landscape or portrait orientation, this controls the direction of the scale and how the timeline is laid out. This can be set when creating a new timeline or from timeline settings.

Page Size

A PDF timeline can be created on one of the following page sizes A5, A4 or A5. This can be set when creating a new timeline or from timeline settings.

Example 1

In this example the orientation has been set to landscape and the page size as been set to A4 (210mm by 297mm). The title and description are at the top, with the scale and events under them. The scale runs from left to right and the PDF gets wider as the scale gets longer. The timeline PDF's height will equal the smaller dimension from that page size (210mm) and its width will depend on length of scale.

Example 2

In this example the orientation has been set to portrait and the page size has been set to A4 (210mm by 297mm). The title and description are at the top, with the scale and events under them. The scale runs from top to bottom and the PDF gets taller as the scale gets longer. The timeline PDF's width will equal the smaller dimension from that page size (210mm) and its height will depend on the length of scale.
