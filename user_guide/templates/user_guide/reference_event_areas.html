    Introduction

    Event areas are used to group events together and position them on the timeline. If you have multiple categories of events that you want to keep separate on a PDF timeline, you could achieve this using an event area for each category.

    On a PDF timeline they effectively make a row or column depending on the timeline's orientation that runs parallel to the scale for all events with that event area to be placed in. Multiple event areas can be added to a timeline then each event can be assigned to a single event area.

    An event must be in an event area to be displayed, therefore at least one event area needs to be created to to display any events.

    Each event area has the following in common...

    • Name
    • Page Position
    • Weight

    Name

    A name which is used to identify the event area in TPlot.

    Page Position

    Controls where to place the event area on the PDF timeline relative to the scale and other event areas. The smaller the page position the closer it will be to the bottom of the page on a landscape timeline and to the left of the page on a portrait timeline.

    See the getting started guide for scale & event area positioning for more details.

    Weight

    Controls the size of the event area on the PDF timeline relative to the weight of the other event areas. On the timeline there is a set amount of space for events to be displayed in, the greater the weight the more of this space an event area will have available to display events.

    See the getting started guide for event area sizing for more details.
