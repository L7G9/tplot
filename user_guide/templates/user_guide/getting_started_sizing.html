Introduction

Each event area has a weight setting, this is used to control the size of each event area in relation to each other. An event area weight can be set when creating a new event area or by editing the event area.

Example 1

Event area A’s weight is 1 and event area B’s weight is 1..

When landscape, event area A is the same height as event area B because its weight is the same. Both are as wide as the scale.

When portrait, event area A is the same width as event area B because its weight is the same. Both are as high as the scale.

Example 2

Event area A’s weight is 2 and event area B’s weight is 1.

When landscape, event area A is twice as high as event area B because its weight is twice as big. Both are as wide as the scale.

When portrait, event area A is twice as wide as event area B because its weight is twice as big. Both are as high as the scale.
