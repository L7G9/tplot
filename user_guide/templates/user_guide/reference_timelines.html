    Introduction

    In TPlot a timeline is stored as a list of events and settings, using these the app can create a perfectly scaled PDF displaying the timeline.

    Each timeline has the following in common...

    • Title
    • Description
    • Scale Unit
    • Scale Unit Length
    • Orientation
    • Page Size
    • Scale Page Position
    • Events
    • Event Areas
    • Tags

    Title

    A title which is used to identify the timeline in TPlot and displayed as a heading when a PDF timeline is created.

    Description

    An optional description which can be used to add additional details about a timeline.

    Is displayed under the title when a PDF timeline is created.

    Scale Unit

    Controls the amount of time between each unit on the scale of a PDF timeline. Because each type of timeline uses different time units this setting varies between the different timeline types.

    See the getting started guide for timeline scales for more details.

    Scale Unit Length

    Controls the distance in CM between each scale unit on the scale of a PDF timeline.

    See the getting started guide for timeline scales for more details.

    Orientation

    Set the page orientation of a PDF timeline to landscape or portrait.

    See the getting started guide for orientation & page size for more details.

    Page Size

    Set the page size of a PDF timeline to A5, A4 or A5.

    See the getting started guide for orientation & page size for more details.

    Scale Page Position

    Controls where to place the scale on the PDF Timeline relative to the event areas. The smaller the page position the closer it will be to the bottom of the page on a landscape timeline and to the left of the page on a portrait timeline.

    See the getting started guide for event area positioning for more details.

    Events

    Events to display on the timeline.

    See the reference guide for events for more details.

    Event Areas

    Event areas for organizing and displaying events in groups.

    See the reference guide for event areas for more details.

    Tags

    Tags for adding category information to events.

    See the reference guide for tags for more details.
