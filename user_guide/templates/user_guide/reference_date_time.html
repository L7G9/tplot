    Introduction

    Date & time timelines have events ordered by time units from the calendar and clock including years, months, weeks, days, hours, minutes and seconds.

    For example a date & time timeline recording the history of space exploration would have events like this, “1969-07-21 02:56 - Neil Armstrong became the first person to walk on the moon”.

    The range time units available in a date & time timeline gives a lot of flexibility, enabling you to create timelines using any of the above time units.

    For example a date & time timeline recording your day would have events like this, “8:05 - Left home for work.”

    See the getting started guide for date & timelines for more details.

    Date & Time Events

    Events in a date & time timeline use this format 2000-04-12 05:30:00. Entering 2000-04-12 would also be acceptable and be equivalent to 2000-04-12 00:0:00.

    Display Formats

    Date & time timelines have two additional settings, scale display format and event display format. These are used to customize how the date and time are displayed on a PDF timeline's scale and events.

    For example if creating a date & timeline focused on events happening on different days over many years, the scale display format could be set to display the year on the scale “1960” and the event display format could be set to display the year, month and day on each event “1969-07-21”
