In this guide we are going to walk through how to create an age timeline using TPlot. An age timeline is a timeline that has events ordered by age in years and months, for example “1 Year and 2 months Jane took her first steps” or “3 Years to 5 Years Company developed new product”.

The information we are going to use are the developmental milestones of a child.

Milestone
At 1 month Emily started opening and closing her hands.
At 2 months Emily started to put her hands in mouth.
At 2 months we noticed Emily's first smile.
At 3 months Emily looks at you when you talk to her.
At 3 months Emily smiles when family and friends talk to her.
At 4 months Emily can focus on her favorite soft toy.
At 4 months Emily gets visibly excited when hearing Mum and Dad's voices.
At 5 months Emily started putting objects in her mouth.
Between 6 and 9 months Emily learned to sit unsupported for several minutes.
At 8 months Emily starts enjoying giving and receiving hugs.

Between 1 Year and 1 Year 5 Months Emily learned to walk starting by using the sofa for support then progressed to walking unaided.
At 1 Year and 8 months Emily can point to people and toys when asked.

Make sure you have registered and are logged in.

From the user timelines page you'll see all the timelines you have created so far and the options to create new timelines. Select the new age timeline option from the side menu.

In the title box enter “Early years development.” A title is required, it is used to identify the timeline in TPlot and as a main heading on the PDF of the timeline. In the description box, enter “Emily Hunter's development years 0-2.” This optional description is used to add additional information about the timeline.

There are two settings for the scale, these are scale unit and scale length. The scale unit determines how many years between each unit marked on the scale of the PDF timeline. The scale unit length determines the distance between these units. Set these to “1 year” and “20 cm” respectively. We are using 1 year every 20 cm as the scale because there are lots of events over a small range of years.

There are three settings for page layout, these are page size, page orientation and scale position. Set these to “A4”, “Landscape” and 0 respectively. Page size and orientation are self-explanatory. Scale position is used to position the scale on the PDF relative to the timeline’s event areas. Leaving it as 0 will ensure the scale is placed at the bottom of the page with all the events above it.

Select the add button to create the new timeline.

All these settings can be edited later or the timeline deleted by selecting the timeline settings option on from the side menu.

For any events to appear on the PDF of the timeline they need to be placed in an event area. Select the new event area option from the side bar.

Enter “Main” as the name, leave page position and weight as 0 and 1 respectively. This will create a single event area above the scale which will contain all our events.

Select the add button to create the new event area.

The new event area will be displayed in a list on age timeline page and can be edited or deleted by selecting it from the the list.

{% endblock event_area_content %} {% block event_content %}

Now we are ready to add our first Age Event, select the New Event option on the side menu.

The 1st event in the table above is “At 1 month Emily started opening and closing her hands.”

In the title box enter "Started opening and closing her hands", leave the description box empty because we have no further information to add.

Set start year to 0 and start month to 1 to represent the age that this event occurred.

Leave has end, end year and end month alone because this event only has a start age.

Select "Main" from the list of available event areas.

Select add to create the event.

The event area will be displayed in a list on age timeline page and can be edited or deleted by selecting it from the the list.

Some of the events in the table above occur over a period of time, for example “Between 6 and 9 months Emily learned to sit unsupported for several minutes.”

These can be added using "has end", "end year" and "end month"

In the title box enter "Learned to sit unsupported for several minutes", leave the description box empty.

Set start year to 0 and start month to 6 to represent the age that this event started.

Tick the has end option so TPlot knows this event has a start and an end.

Set end year to 0 and end month to 9 to represent the age that this event end.

Select "Main" from the list of available event areas.

Select add to create the event.
