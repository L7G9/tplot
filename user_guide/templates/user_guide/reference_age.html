    Introduction

    Age timelines have events ordered by age in years and months.

    For example an age timeline recording a child's milestones would have events like this, "1 year and 2 months - Nina took her first steps."

    See the getting started guide for age timelines for more details.

    Age Events

    Events in an age timeline use a single number for both year and a month, these can be negative.
