In this guide we are going to walk through how to create a date & time timeline using TPlot. A date & time timeline has events ordered by units from the calendar and clock, for example “1969-7-20 - Neil Armstrong became the 1st person to walk on the moon.” or “6:50 - I woke up.” Date & time timelines can use time units including years, months, weeks, days, hours, minutes and seconds, the following guide will show you how to make a timeline using years only.

The topic we are going to use in guide is the history of aviation.

Make sure you have registered and are logged in.

From the user timelines page you'll see all the timelines you have created so far and the options to create new timelines. Select the new date & time timeline option from the side menu.

In the title box enter “History of Flight.” A title is required, it is used to identify the timeline in TPlot and as a main heading on the PDF of the timeline. In the description box, enter “By John Doe.” This optional description is used to add additional information about the timeline.

There are three settings for the scale, these are scale unit, scale length and scale display format. The scale unit determines how many date or time units are between each unit marked on the scale of the PDF timeline. The scale unit length determines the distance between these units. The scale display format determines the which time units from the calendar and clock are used on scale. Set these to “10 years”, “3cm” and "2000 (year)" respectively. In this example we are using 10 years every 3 cm as the scale because in the list of aviation events there are a few events every decade. The scale display format is set to display year only because our events only have a year and no other information.

There are three settings for page layout, page size, page orientation and scale position. Page size and orientation are self-explanatory. Scale position is used to position the scale on the PDF relative to the timeline's event areas. Set these to “A4”, “Landscape” and 0 respectively. Leaving scale position as 0 will place the scale at the bottom of the page with all the events above it.

There is a final setting called "Event display format". This determines the which time units from the calendar and clock are used on all events." Set this to "2000 (year)" again because our events only have a year and no other time units.

Select the add button to create the new timeline.

All these settings can be edited later or the timeline deleted by selecting the timeline settings option on from the side menu.

For any events to appear on the PDF of the timeline they need to be placed in an event area. Select the new event area option from the side bar.

Enter “Main” as the name, leave page position and weight as 0 and 1 respectively. This will create a single event area above the scale which will contain all our events.

Select the add button to create the new event area.

The new event area will be displayed in a list on age timeline page and can be edited or deleted by selecting it from the the list.

Now we are ready to add our first event, select the new event option on the side menu.

The 1st event in the table above is “1783 1st free aerial voyage in a hot air balloon, Jean Francois Pilatre Dr Rozier and Marquis d'Arlandesin Montgolfier.” In the title box enter "1st free aerial voyage in a hot air balloon". In the description box enter "Jean Francois Pilatre Dr Rozier and Marquis d'Arlandesin Montgolfier". Set start date time to "1783-01-01" to the year this event occurred. We are purposely setting the month and day as the January 1st and omitting the time because we don't have that information in our table. This will cause the event to be positioned at the start of the year 1783 on the scale of the PDF timeline which is good enough for this example. We could set the date and time to the nearest second and TPlot would be able to position the event accurately. Leave has end, end date time alone because this event only has a start date time. Select "Main" from the list of available event areas.

Select add to create the event.

The event area will be displayed in a list on date & time timeline page and can be edited or deleted by selecting it from the list.

In the description box enter "Starts with the launch of Sputnik 1, the 1st man-made earth satellite and ends with Neil Armstrong being the 1st person to walk on the moon." Set start date time to "1957-01-01", then tick the has end option and set the end date time to "1969-01-01". Select "Main" from the list of available event areas.
